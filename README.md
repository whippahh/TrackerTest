# ProgressScape

An OSRS progression tracker and planner. Track your quests, achievement diaries, boss kill counts and collection log — all in one place.

🔗 **Live site:** [whippahh.github.io/ProgressScape](https://whippahh.github.io/ProgressScape/)

## Features

- Quest completion tracker
- Achievement Diary progress (Easy / Medium / Hard / Elite)
- Boss kill count tracking
- Collection log sync
- Share your progression with a link

## RuneLite Plugin

Data is synced automatically via the **ProgressScape RuneLite plugin**.

- Install it from the RuneLite Plugin Hub — search **ProgressScape**
- Plugin repo: [github.com/whippahh/ProgressScape-plugin](https://github.com/whippahh/ProgressScape-plugin)

Once installed, your data syncs on login/logout. Collection log requires a manual sync with the log open.
